HTML

  • The head element is a container for metadata.
  • Meta elements contain information about the page that is used by the browser, and not visible to the user.
  • Use semantic elements where possible. Semantic elements indicate the role of their content.
  • "Element" refers to the entire element from opening tag to closing tag, whereas "tag" only refers to what's inside the angle brackets.
  • Alt text is used for screen readers and assistive technologies, as well as search engine rankings.

CSS

  • The margin attribute controls how much space is outside of an element, whereas the padding attribute controls the amount of space inside an element.
  • display: block makes each image start on a new line.
  • Using the wildcard selector (*) selects all elements.
  • Preceding a selector with . indicates it is a class.

Git

  • git clone [SSH key]: clones repo to local machine
  • git pull origin main: pulls latest from main
  • git status: checks what branch you're currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git commit -m "[message]": commits changes with a message
  • git add -A: stages all changes before pushing
  • It's good practice to pull latest main before pushing
  • git push origin branch-name: pushes changes

JavaScript

  • JavaScript is a dynamic language, so variables are not directly associated with any particular value type, and can be re-assigned values of all types.
  • JavaScript is weakly typed, so it allows implicit type conversion instead of throwing type errors.
  • Null, Undefined, Boolean, Number, BigInt, String, and Symbol are all primitive types in JavaScript.
  • An array is a variable that holds a group of data. The data can be different types. They are zero-indexed.
  • Example of defining an array: var arrayName = [val1, "val2", val3]
  • A for loop is a convenient way to perform the same operation multiple times. They pair well with Arrays because you can use a for loop to perform the same operation on every value in the array.
  • Example of for loop formatting: for ( var x = 0; x < array.length; x++) {code block}
  • Control flow is the order in which a computer executes code in a script. JavaScript executes from top to bottom.
  • We can use if statements to control when certain pieces of code are triggered or skipped.
  • A function is a reusable block of code which only executes when it is called.
  • Example of defining a function: function functionName() {code block}
